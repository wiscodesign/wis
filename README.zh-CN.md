<h1 align="center">
  Wis Design System
</h1>

Wis 是 Wis.design 的开源设计系统，旨在为B端产品提供卓越的用户体验。本系统以基于本质的设计语言为基础，提供包含可运行的代码、设计工具、产品设计指南。

我们是一群热衷于开源的设计师和开发者，我们渴望打造一个充满活力的贡献者社区。
