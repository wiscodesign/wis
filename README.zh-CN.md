<h1 align="center">
  Wis Design System
</h1>

Wis 是 [Wis.design](https://wis.design) 推出的开源设计系统，致力于为 B 端产品提供卓越的多设备用户体验

>所有组件均支持 **PC** 与 **Mobile** 双端，属性与功能保持一致，用户体验经过系统性设计和验证

系统基于“本质驱动”的设计语言，提供可运行的代码组件、设计工具与产品设计指南，帮助团队更高效地构建统一、可扩展的界面
